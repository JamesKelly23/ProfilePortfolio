Projects

  • Schedule Manager - ASP.NET Core 6.0 MVC Web Application

    • Functional Demo
    • Code Repository - Github
    • This application was designed for use by the management team and crew of a restaurant to manage and view their schedules, availabilities, and time off requests. The application also allows the users to trade shifts with other employees.
    • Developed using the Agile framework and a scrum team of four, led by myself as the scrum master
    • Uses modern MVC framework to serve dynamic content
    • Application interfaces with Microsoft SQL Server 2019
    • Program uses the Razor Pages framework using CSHTML files to generate content
    • All database write interactions are facilitated through stored procedures for security
    • SSL facilitates a secure connection to the server through Cloudflare with End to End encryption enabled
    • Data access is facilitated through Database Context classes to prevent errors and separate data access from the controller and view layers
  • D2L Hourly Upload Script

    • Code Repository - Github
    • This script facilitates EDI communications between a college's Main ERP system to a third party distance learning application to keep them synchronized
    • Coding was done using Microsoft Powershell, and uses WinSCP to facilitate the SFTP upload
    • The script is fully configurable via a .INI file that is read upon script start
    • The script archives all files sent to ensure there is a backup in case of issues
    • The transcript of each run is also stored with each archive, and is emailed to a list of users specified in the configuration file
    • When multiple batches are generated between runs (script is designed to be run on an hourly basis), the script sorts them and uploads them in the proper order to prevent data dependency issues
    • When the script encounters an error, whether it is a file permission error or an upload error, the script aborts, and sends an email alert along with the transcript of the run to a list of IT Support personnel configured in the INI file.
